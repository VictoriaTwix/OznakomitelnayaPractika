-# SimpleAnalysis

Проект по анализу данных, выполненный в рамках ознакомительной практики в ООО «Апрель ИТ Проект». Реализован полный цикл обработки числовых данных: генерация, очистка, статистический анализ и визуализация.

## 📋 Описание

Программа генерирует массив из 1000 случайных целых чисел в диапазоне [-10000, 10000], выполняет его очистку от «цифрового мусора», рассчитывает основные статистические характеристики и строит наглядные графики распределения.

### Основные возможности:
- Генерация случайных данных с фиксированным seed для воспроизводимости
- Очистка данных от пропусков, бесконечных значений и выбросов
- Расчёт статистических характеристик (min, max, сумма, std, дубликаты)
- Сортировка данных по возрастанию и убыванию
- Визуализация в виде линейных графиков и гистограмм
- Сохранение результатов в CSV-файлы

## 🛠 Технологии

- Python 3.12+
- Pandas — обработка и анализ данных
- NumPy — высокопроизводительные математические вычисления
- Matplotlib — визуализация данных
