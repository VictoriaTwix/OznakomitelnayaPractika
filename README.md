## О проекте

Проект **SimpleAnalysis** разработан в рамках учебной ознакомительной практики в отделе внедрения компании «Апрель ИТ Проект». Его цель - закрепить теоретические знания по анализу данных на практике и освоить полный цикл работы с числовыми массивами с использованием языка Python и специализированных библиотек.

Программа моделирует реальную аналитическую задачу: от генерации «сырых» данных до формирования статистических выводов и визуализации. Такой подход позволяет на практике отработать навыки, востребованные в профессии бизнес-аналитика.

### Что делает программа:
- Генерирует выборку из 1000 случайных целых чисел
- Проверяет данные на наличие пропусков, выбросов и некорректных значений
- Выполняет очистку от «цифрового мусора»
- Рассчитывает ключевые статистические характеристики
- Сортирует массив по возрастанию и убыванию
- Строит линейные графики и гистограммы распределения
- Сохраняет промежуточные и итоговые результаты в CSV-формате

## Структура проекта и описание файлов

Ниже приведено описание всех файлов, входящих в состав репозитория:

| Файл | Описание |
|------|----------|
| **`SolovevaVA.py`** | Основной исполняемый скрипт. Содержит полный код программы: генерацию данных, очистку, расчёт статистик, визуализацию и сохранение результатов. |
| **`simple_analysis_data.csv`** | Промежуточный CSV-файл, содержащий исходный набор данных (столбец `Original_Data`). Создаётся на этапе сохранения данных после первичной очистки. |
| **`simple_analysis_dataframe.csv`** | Итоговый CSV-файл с расширенным DataFrame. Включает три столбца: `Original_Data` (исходные данные), `sorted_asc` (сортировка по возрастанию) и `sorted_desc` (сортировка по убыванию). |
| **`Графики.png`** | Комбинированный график (линейный график исходных данных + гистограмма с округлением до сотен). Сохраняется как `visualization_1.png`. |
| **`СортировкаЗначенийГрафик.png`** | График с наложением двух рядов: отсортированных по возрастанию (зелёная линия) и по убыванию (красная линия). Сохраняется как `visualization_2.png`. |
| **`README.md`** | Текущий файл с описанием проекта, инструкцией по установке и запуску, а также пояснением структуры репозитория. |
| **`СолвьеваВА_Ознакомительная Практика_отчет_ред.2.docx`** | Полный отчёт о прохождении практики. Содержит введение, обоснование выбора инструментов, описание алгоритмов, результаты анализа, выводы и список литературы. Оформлен в соответствии с требованиями ГОСТ. |
